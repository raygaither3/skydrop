# SkyDrop 🚁

SkyDrop is a drone delivery dispatch platform built with Flask.

## Features

- Address → GPS conversion
- Interactive map with draggable delivery pin
- Drone assignment based on distance, battery, and payload
- Mission tracking with real-time route visualization

## Tech Stack

- Python / Flask
- Leaflet.js (maps)
- SQLite (database)
- Geopy (distance + geocoding)

## Vision

SkyDrop is designed as a foundation for real-world drone delivery systems, including:
- automated dispatching
- waypoint generation
- real-time drone telemetry
- secure delivery systems
